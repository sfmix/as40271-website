## What we collect

This site is a static Hugo site hosted on GitHub Pages. We don't use analytics, tracking pixels, or ad scripts. We don't set cookies. We're not on that.

## Third-party services

- **GitHub Pages** — serves the site. GitHub may log IP addresses and user-agent strings in line with their own [privacy statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## Fonts

All web fonts are self-hosted. No requests go out to Google Fonts or any other third-party font services — locked in.
