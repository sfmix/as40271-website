## Was wir erfassen

Diese Website ist eine statische Hugo-Website, die auf GitHub Pages gehostet wird. Wir verwenden keine Analyse-Tools, Tracking-Pixel oder Werbe-Skripte. Wir setzen keine Cookies.

## Drittanbieter-Dienste

- **GitHub Pages** — stellt die Website bereit. GitHub kann IP-Adressen und User-Agent-Strings gemäß der eigenen [Datenschutzerklärung](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) protokollieren.

## Schriftarten

Alle Web-Schriftarten werden selbst gehostet. Es werden keine Anfragen an Google Fonts oder andere Schriftarten-Dienste von Drittanbietern gesendet.
