## Qué recopilamos

Este sitio es un sitio estático de Hugo alojado en GitHub Pages. No usamos analíticas, píxeles de seguimiento ni scripts publicitarios. No establecemos cookies.

## Servicios de terceros

- **GitHub Pages** — sirve el sitio. GitHub puede registrar direcciones IP y cadenas de user-agent de acuerdo con su propia [declaración de privacidad](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## Tipografías

Todas las tipografías web están autoalojadas. No se hacen solicitudes a Google Fonts ni a otros servicios de tipografías de terceros.

## Contacto

Si tienes preguntas sobre esta política, escribe a [tech-c@sfmix.org](mailto:tech-c@sfmix.org).
